# Инструкция по работе с IG Parser

## Что это?

IG Parser — это бот, который отслеживает Instagram-аккаунты, находит посты про научные исследования и выгружает их в Google-таблицу. Рекламные и личные посты пропускаются, в выгрузку попадает только экспертный контент.

**Что делает бот:**
- Берёт посты из указанных Instagram-аккаунтов
- Находит упоминания статей в PubMed (PMID в подписи, скриншоты, транскрипт Reels)
- Добавляет явно цитируемые исследования; при наличии основных добавляет до 3 похожих из PubMed
- Формирует краткое содержание и теги
- Сохраняет всё в Google-таблицу

---

## Как пользоваться

### 1. Добавьте аккаунты

В поле **«Блогер(ы) (через запятую)»** введите Instagram-имена без `@`, через запятую.

Пример: `dangarnernutrition, skerco00`

### 2. Сохраните список

Нажмите кнопку **«Сохранить аккаунты для мониторинга»**.

Без этого авто-выгрузка не будет работать: бот не знает, какие аккаунты отслеживать.

### 3. Автоматическая выгрузка

Выгрузка запускается **каждый день в 8:00 по Москве** без вашего участия.

Все результаты попадают в Google-таблицу. Ссылку на таблицу вы увидите на странице.

### 4. Ручная выгрузка

Чтобы выгрузить посты сразу, не дожидаясь 8:00, нажмите **«Выгрузить новые посты сейчас»**.

Обрабатываются только **новые** посты (с момента последней выгрузки), дубликаты не создаются.

### 5. Просмотр результатов

Нажмите ссылку **«открыть»** рядом с текстом про таблицу — откроется Google-таблица со всеми выгруженными данными.

## Структура таблицы

Колонки должны быть в точном порядке (именно эти названия):

| № | Колонка | Описание |
|---|---------|----------|
| 1 | Название поста | Тема или заголовок поста |
| 2 | Инстаграм | Имя аккаунта автора |
| 3 | Дата публикации | Когда опубликован пост |
| 4 | Ссылка на пост в инстаграм | Прямая ссылка на пост |
| 5 | Тип контента | reel, carousel или post |
| 6 | Описание | Подпись поста (caption) |
| 7 | Картинка | URL превью/изображения |
| 8 | Транскрипт | Текст из Reels (если есть) |
| 9 | Саммари | Краткое содержание |
| 10 | Название исследования | Статья из PubMed |
| 11 | Автор | Автор исследования |
| 12 | Год исследования | Год публикации |
| 13 | Ссылка на исследование PMID | Ссылка в PubMed |
| 14 | Ссылка на полный текст исследования | Полный текст статьи |
| 15 | Тег по смыслу исследования | Ключевые слова |
| 16 | Источник цитаты | описание / картинка / транскрипт / похожее |

Для постов с несколькими исследованиями блок колонок 1–9 повторяется пустыми, меняются только 10–16.

---

## Сколько постов обрабатывается?

- При ручной выгрузке: до 20 постов с **каждого** аккаунта
- При авто-выгрузке в 8:00: только посты, которые появились после прошлой выгрузки

---

## Частые вопросы

**Почему пост не попал в таблицу?**  
Бот пропускает личные посты, рекламу и контент без ссылок на исследования. Пост должен содержать PMID, скриншот PubMed или явное упоминание исследования.

**Можно ли добавить больше 20 постов при первом запуске?**  
По умолчанию — до 20 с каждого аккаунта. Дальнейшие посты попадут при следующих выгрузках.

**Как часто обновляется таблица?**  
Автоматически — раз в день в 8:00 по Москве. Вручную — в любой момент по кнопке.
